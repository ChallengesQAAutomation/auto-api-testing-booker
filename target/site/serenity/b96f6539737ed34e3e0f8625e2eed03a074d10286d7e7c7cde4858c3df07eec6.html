Servicio Web De Gestion De Reservas

 Happy Paths (tag)

Creacion, modificacion, busqueda y eliminacion de la misma reserva
 

Cast

  Administrador del gestor de reservas

Administrador del gestor de reservas can:
  • call an API at https://restful-booker.herokuapp.com

Scenario Outline

Given que el usuario admin crea una reserva para un cliente llega el "<checkin>" y se va el "<checkout>" con un "<additionalneeds>"
When el usuario admin corrige el "<name>" del cliente
Then el sistema debera de traer la reserva cuando la busca con filtro de nombre
And usando el BookingId
Then el usuario admin elimina la reserva

Examples: 

# Checkin Checkout Additionalneeds Name
1 2023-02-02 2023-02-12 BreakFast Joaquin
2 2021-12-02 2023-11-12 Rent Car Lorenzo
3 2022-21-02 2023-02-10 Pool Arya
